Fixing Invalid CSS

Instructions

  1. Open index.html in your browser and observe the styles.
  2. Open css/style.css in a text editor.
  3. Visit the W3C CSS Validator.
  4. Copy and paste the CSS code into the validator.
  5. Click "Check" and fix the errors in style.css.
  6. Repeat until the validator shows "Congratulations! No Error Found."
  7. Check your progress by refreshing index.html in your browser.
